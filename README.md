# 🛠️ Telegram Consumables Bot (aiogram 3)

Простой и эффективный Telegram-бот для механиков, который автоматизирует учет расходных материалов. Бот принимает название и количество расходника, после чего мгновенно записывает данные в Google Таблицу.

## ✨ Основные функции
- **Автоматизация**: Забудьте о ручном заполнении таблиц.
- **Идентификация**: Бот автоматически берет имя механика из профиля Telegram.
- **Простота**: Общение в формате "Название Количество".
- **Google Sheets**: Интеграция через официальный API (gspread).

## 🚀 Установка

1.  **Клонируйте репозиторий**:
    ```bash
    git clone <url-вашего-репозитория>
    cd <папка-проекта>
    ```
2.  **Установите зависимости**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Настройте переменные окружения**:
    - Создайте бота в [@BotFather](https://t.me/BotFather) и получите API-токен.
    - Скопируйте файл `.env.example` в `.env` и впишите свои данные:
      ```env
      BOT_TOKEN=ваш_токен_здесь
      SPREADSHEET_ID=id_вашей_таблицы
      CREDENTIALS_FILE=service_account.json
      ```

---

## 📊 Настройка Google Sheets API

Для работы бота необходимо создать сервисный аккаунт в Google Cloud:

1.  **Google Cloud Console**:
    - Перейдите в [Google Cloud Console](https://console.cloud.google.com/).
    - Создайте проект и включите **Google Sheets API** и **Google Drive API**.
2.  **Сервисный аккаунт**:
    - Создайте "Service Account" в разделе Credentials.
    - Выпустите JSON-ключ, переименуйте его в `service_account.json` и поместите в корень проекта.
3.  **Доступ к таблице**:
    - Скопируйте `client_email` из JSON-файла.
    - Откройте вашу Google Таблицу и нажмите **"Поделиться"**.
    - Добавьте этот email с правами **"Редактор"**.

## 📝 Пример использования

Механик пишет боту:
> `Масло 5`

Бот отвечает:
> ✅ **Записано!**
> 🔧 *Иван Иванов*
> 📦 *Масло: 5*

В Google Таблице появится строка:
| Дата | Имя механика | Расходник | Количество |
|------|--------------|-----------|------------|
| 20.03.2026 19:25 | Иван Иванов | Масло | 5 |
